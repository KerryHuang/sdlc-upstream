```markdown

3. 使用者點擊「儲存」
   - 系統行為：全欄位驗證 → 交易開始 → {SQL} → 交易結束
   - 成功：{回應}
   - 失敗：{錯誤訊息}
```

| 操作 | 觸發事件順序 | 涉及程序 | 資料表操作 |
|------|------------|---------|-----------|
| 新增 | cmdAdd→Valid(多個)→cmdSave→p_save | {列表} | INSERT INTO {table} |

| 刪除 | 選取→cmdDel→關聯檢查→確認→p_delete | {列表} | DELETE/DEL_MARK |
| 查詢 | 條件→cmdFind→p_load/display | {列表} | SELECT FROM {table} |

```sql
-- 列舉值

-- 編號格式

-- 日期範圍
SELECT MIN({date_col}), MAX({date_col}) FROM {table}
-- 金額/數量範圍
```
